Run The Bot
To run cordbot on your own server:

Clone this repo.
Install dependencies with npm install.
Get the required API keys and fill them in.
npm test

Keys or Token:
https://discord.com/login?redirect_to=%2Fdevelopers%2Fapplications%2F
